## Prometheus brownbag

Introduction for Prometheus monitoring system.

## How to use presentation:

Prerequireties:

    1. install Golang
    2. install Golang present-plus tool
    3. pull this repo to you PC


1. Download:
    ```sh
    $ go get -u github.com/davelaursen/present-plus
    ``` 
    
2. Run presentation:
    ```sh
        $ present-plus
    ```
3. Open in browser next link:
    ```sh
    http://localhost:4999
    ```
